# Gestion de Riesgos de Maquinas Industriales
Esta aplicacion permite gestionar los riesgos asociados a las maquinas industriales en una planta de produccion. Proporciona herramientas para identificar, evaluar y mitigar riesgos, asi como para monitorear el estado de las maquinas en tiempo real.

## NOTA:
Este proyecto es un prototipo y no debe ser utilizado en entornos de produccion sin una evaluacion y adaptacion adecuada a las necesidades especificas de la planta y las normativas vigentes.

## REQUISITOS DEL SISTEMA
- Python 3.8 o superior
- Bibliotecas requeridas: ttkbootstrap, pandas, numpy, matplotlib

## INSTALACION
1. Clonar el repositorio
2. Crear un entorno virtual
3. Instalar las dependencias

```bash
git clone <repositorio>
cd <repositorio>
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## USO
```python
python app.py
```
